Description:
Ce projet dans le cadre du cours de Programmation et base de données du semestre 7 de Master 1 SCA, à pour but final de réfléchir à la conception et à la réalisation d'un système d'information capable de gérer les notes du baccalauréat.

Installation:
 - requis:
   1) Java (J2SE) version 1.8 ou + (https://www.oracle.com/fr/java/index.html)
   A partir du dossier racine :
 - Linux : ./install.sh
 - Windows: install

Pour plus d'informations voir ./doc/index.html
