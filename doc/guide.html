Le dossier /bin contient plusieurs scripts (candidat.sh par exemple) pour importer les données des fichiers .csv dans la base de données. Vous devez d'abord compiler vos classes .java : 

					%cd <install directory>
					%./install.sh (ou install sur Windows)
Ensuite, vous devez exécuter chaque script (sauf template) se trouvant dans le dossier /bin en ayant changer au préalable les paramètres de la commande avec vos paramètres : 
					%cd bin
Ouvrez chaque script pour remplacer les paramètres : 
					PARAMS="-i ../samples/<nom-fichier> -p <mot-de-passe> -r <uri-bdd> -u <utilisateur-bdd>"
Et enfin, exécutez les : 
					%./candidat.sh (ou candidat sur Windows)
A ce stade toutes les données contenues dans les fichiers .csv seront contenues dans la base de données créée dans la section "Installation".
